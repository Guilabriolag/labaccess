Como Funciona

Guia rápido do sistema LabAccess

1️⃣
Preencha os dados do colaborador Informe nome, empresa, cargo e demais informações. Utilize os toggles ao lado de cada campo para controlar o que será exibido no crachá.
2️⃣
Personalize o visual Escolha a cor de destaque do crachá e selecione o plano de acesso que deseja simular.
3️⃣
Gere o QR Code de acesso Clique no botão "GERAR QR CODE" para criar um código único contendo todas as informações do colaborador.
4️⃣
Simule o pagamento Acesse a aba "Simular Pagamento" e clique no botão para visualizar o modal de PIX demonstrativo.
5️⃣
Baixe o crachá Após a simulação, o crachá será baixado automaticamente em formato PNG para impressão ou uso digital.
